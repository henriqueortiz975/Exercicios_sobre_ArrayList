# Exercícios sobre ArrayList

1. Criando um ArrayList
Crie um programa que declare um ArrayList de números inteiros e adicione cinco valores informados pelo usuário. Em seguida, exiba todos os elementos armazenados.

2. Removendo elementos
Crie um ArrayList de strings e adicione os seguintes nomes: "Ana", "Carlos", "Bruna", "Daniel", "Eduardo". Peça ao usuário um nome para remover e, se existir na lista, remova-o e exiba a lista atualizada.

3. Ordenando um ArrayList
Crie um ArrayList de números inteiros e adicione dez valores aleatórios entre 1 e 100. Em seguida, ordene os valores em ordem crescente e exiba a lista ordenada.

4. Buscando um elemento
Crie um ArrayList de nomes de produtos e permita que o usuário pesquise se um determinado produto está na lista. Informe se o produto foi encontrado ou não.

5. Substituindo um elemento
Crie um ArrayList de cinco cidades e exiba a lista. Depois, peça ao usuário o nome de uma cidade para substituir e uma nova cidade para colocar no lugar. Atualize o ArrayList e exiba a lista modificada.

6. Contando ocorrências
Crie um ArrayList onde o usuário pode inserir várias palavras. Permita que ele digite uma palavra específica para contar quantas vezes ela aparece na lista.

7. Removendo elementos repetidos
Crie um ArrayList com alguns números inteiros, incluindo valores duplicados. Remova os valores duplicados e exiba a lista sem repetições.

8. Invertendo a ordem dos elementos
Crie um ArrayList com seis elementos e exiba-os na ordem original e, depois, na ordem inversa.

9. Copiando um ArrayList
Crie um ArrayList de nomes e copie todos os elementos para um novo ArrayList. Exiba os dois ArrayLists para confirmar a cópia.

10. Usando Iterator para percorrer um ArrayList
Crie um ArrayList de cinco números inteiros. Utilize um Iterator para percorrer a lista e exibir os elementos.

11. Somando valores de um ArrayList
Crie um ArrayList de números decimais e calcule a soma de todos os valores armazenados.

12. Encontrando o maior e o menor valor
Crie um ArrayList de números inteiros e exiba o maior e o menor valor da lista.

13. Mesclando dois ArrayLists
Crie dois ArrayLists de números inteiros com cinco elementos cada. Crie um terceiro ArrayList que contenha todos os elementos dos dois primeiros e exiba o resultado.

14. Criando um menu interativo com ArrayList
Crie um programa que utilize um ArrayList para armazenar uma lista de tarefas. O usuário deve poder adicionar, remover e listar as tarefas através de um menu interativo no console.

15. Utilizando ArrayList com uma classe personalizada
Crie uma classe Aluno com os atributos nome e nota. Depois, crie um ArrayList<Aluno> e adicione alguns alunos com suas notas. Exiba a lista de alunos e, ao final, mostre a média das notas.
